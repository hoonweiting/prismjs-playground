Prism.js playground!

Here is where I load CSS files to see what each token looks like in a few languages (and with some plugins enabled). Unfortunately, not all tokens are covered in the selected languages, much less all the other languages.

HTML

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8" />
    <title>I can haz embedded CSS and JS</title>
    <style>
      body {
        background: url('foo.png');
        color: #EEE;
        height: calc(100vw - 10px);
      }

      @media screen and (min-width: 100px) {
        p {
          color: rgb(12, 12, 12) !important;
        }
      }
    </style>
  </head>

  <body>
    <h1>I can haz embedded CSS and JS</h1>
    <script>
      if (true) {
        console.log('foo');
      }
    </script>
    <h2 class="cheezborger" style="color: pink">CHEEZBORGER</h2>
  </body>
  <!-- KTHXBAI -->
</html>
